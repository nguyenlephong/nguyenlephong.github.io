# Software Engineer Intern – GreenNode

## 🌟 Overview

As a **Software Engineer Intern at GreenNode**, you will contribute to the design, development, and improvement of **Cloud Platform services** and **AI infrastructure components**.

This role focuses on building strong foundations in:

- Programming
- System thinking
- Engineering best practices

You will work under close mentorship from senior engineers.

> This is a **learning-heavy engineering role**, designed for early-career talents who want to grow in:

- Backend development
- Cloud-native systems
- Large-scale distributed platforms

---

## 🛠️ Job Description

- Write code and build simple to medium backend features under the mentorship of senior engineers
- Participate in the design, development, and implementation of backend services for the GreenNode Cloud Platform
- Assist in developing and optimizing backend applications to ensure:
  - Performance
  - Reliability
  - Scalability
- Contribute to software releases, including:
  - Coding
  - Testing
  - Deployment preparation
- Analyze problems, debug issues, and improve system quality
- Participate in:
  - Code reviews
  - Technical discussions
  - Team rituals
- Continuously explore new technologies and propose improvements for:
  - System architecture
  - Implementation

📍 **Working location:** VNG Campus, Ho Chi Minh City
